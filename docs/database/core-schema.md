# Eraxis Core Schema

## tenants

Stores organizations using Eraxis.

## branches

Stores branches belonging to tenants.

## users

Stores platform users.

## roles

Stores RBAC roles.

## permissions

Stores RBAC permissions.

## user_roles

Maps users to roles.

## role_permissions

Maps roles to permissions.
